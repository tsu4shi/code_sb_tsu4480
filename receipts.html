レシート読み込み

レシート画像をまとめて読み込み、Google Cloud Vision OCRで明細行の表にします。結果は画面で確認・編集でき、CSVとしてダウンロードできます（家計簿ツールとの突合は次バージョン予定）。

※ レシート画像はOCRのため Google Cloud Vision API に送信されます。APIキーはこの端末の localStorage にだけ保存され、リポジトリやサーバーには置きません。ログインは家計簿ツールと同じGoogleアカウント許可リストです。

ログイン

許可されたGoogleアカウントでのみ利用できます。

1. Vision APIキー

Google Cloud で発行した APIキーを入力してください。このブラウザの localStorage に保存されます（サーバーには送られません）。キーには HTTPリファラ制限を付け、Vision の月次クォータを 1000 に設定することを推奨します。

2. レシート画像を読み込む

JPEG / PNG / WebP / GIF の画像を複数選択できます。無料枠（月 1000 枚）を超える分は処理しません。
